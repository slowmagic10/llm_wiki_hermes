# 企业手册模板

适用于员工手册、内部制度、行政规范、部门职责和企业级政策。

```markdown
---
title: "<制度或手册名称>"
type: policy
status: draft
owner: "<制度负责人>"
updated: "YYYY-MM-DD"
domain: "<domain_id>"
customer_safe: false
rag: false
category:
  - enterprise_handbook
tags:
  - internal
  - policy
summary: "<一句话说明制度目的>"
department: "<发布部门>"
audience:
  - "<适用人员或部门>"
effective_date: "YYYY-MM-DD"
review_cycle: "<例如：每年复审>"
confidentiality: internal
sources:
  - type: draft_import
    ref: "待补充正式制度文件或审批记录"
---
```
